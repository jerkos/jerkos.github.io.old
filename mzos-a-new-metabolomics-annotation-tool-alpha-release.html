mzOS, a new metabolomics annotation tool, alpha release

Date dim. 02 novembre 2014 By omics services team Tags python / data-mining / mass-spectrometry / metabolomics

mzOS is an tool for annotating detected features in LC/MS analysis. It aims to be run after XCMS (a famous R package for prepocessing LC/MS data). mzOS is mainly written in python using the very common scientific stack numpy, scipy and scikit-learn. We all know that python is limited by nature in term of execution speed. Thanks to theses packages, Python is one of the most used language for treating scientific data keeping developer productivity at a high level.

Adducts and isotopes labeling

Adducts and isotopes are distinguished from real monoisotopic feature using different clustering techniques, using in particular detected peaks shape and peaks intensity across sample and mz over charge and elution time coordinates.

Database annotation

Each detected feature is search against selected databases (for the moment, HMDB, LMSD, and KEGG) passing considered adduct (usually M+H or M-H) very effectively since each database could be shipped or constructed using a mzOS script under the form of an sqlite file. Several score are calculated. One taking into account the difference between observed isotopic pattern of one feature and a theoritical one derived from the annotation. The other is bayesian approach to determine annotations probabilities given the knowledge of the metabolic network (thanks to KEGG).
